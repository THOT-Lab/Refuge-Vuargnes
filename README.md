# Le refuge des Vuargnes

Site statique de présentation pour la mise en location du chalet **Le refuge des Vuargnes** à Mont-Saxonnex.

## Site en ligne

https://thot-lab.github.io/Refuge-Vuargnes/

## Lancer en local

Ouvrez directement `index.html` dans un navigateur, ou servez le dossier avec un petit serveur local:

```bash
python3 -m http.server 8080
```

Puis ouvrez `http://localhost:8080`.
